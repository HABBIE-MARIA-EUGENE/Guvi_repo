why bootstrap is powerful?

.container = gives proper horizontal padding

.row + col-... = implements responsive colums

.col-12 col-md-6 col-lg-5 = full width on phones , half on tabs , 5/12 on lg screens

spacing utilities: py-5,mb-3,p-4  (helps us instead of writing custom css and all) 

form classes: .form-label + .form-control = gives us consistent UI instantly.


bootstrap utilities

spacing: m-?/p-? (e.g.,mt-3,px-2)

Display: d-flex,justify content center, align-items-center

sizing:w-100, w-50      100= full width ,50 = half

Text:text-center,text-muted,fw-bold

Note: To override bootstrap styles, need to load style.css only after loadinf bootstrap
